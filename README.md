# SEED-Cameroon — Assignment Submissions

This repo is where students submit their **solved exercises**, one folder per track → week → day.

It's structured to mirror the [`curriculum`](https://github.com/SEED-Cameroon/curriculum) repo, which holds the lesson content and exercise instructions. This repo holds only your solutions.

## Tracks

- `backend-track/` — Backend Track (8 weeks)
- `frontend-track/` — Frontend Track (8 weeks)
- `fullstack-track/` — Fullstack Track (8 weeks)
- `onemonth-backend-track/` — One-Month Backend Track (4 weeks)
- `onemonth-frontend-track/` — One-Month Frontend Track (4 weeks)
- `python-optional-track/` — Python Optional Track (8 weeks)

## Workflow

1. Read the day's lesson and exercise list in `curriculum/<track>/<week>/<day>/`.
2. Solve the exercises and add your files under `assignments/<track>/<week>/<day>/` in this repo.
3. Commit and push. Open a PR against `main` if working on a fork.
4. File your daily report separately in the [`daily-reports`](https://github.com/SEED-Cameroon/daily-reports) repo.

## Related repos

- `curriculum` — lesson content, resources, exercise instructions
- `daily-reports` — end-of-day report submissions
